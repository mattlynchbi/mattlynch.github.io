Background

This is a sales dashboard for a fictitious global FX trading business. The underlying data was randomly generated using Python code.
The purpose of the dashboard is to help management track performance of the business and understand client activity.
The Overview page provides a summary of business performance across a selection of key metrics. Charts provide a comparison between current and prior year.
The Region and Product page provides insight into performance by client location and product.
The Client Detail page provides a more detailed summary of activity and performance for a single client.

Power BI Techniques

  • Integrated page navigation bar and backgrounds created using Figma.
  • Integrated filter panel that can be open and closed.
  • Field parameters to change the metrics displayed in visuals based on user selection.
  • The monthly time series bar chart is a custom visual created using Deneb (vega-lite).
  • Dynamic SVG images are used to show the trend lines in the KPI cards and the bullet bars in the table visuals.
